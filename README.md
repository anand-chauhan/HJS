📰 Hindu Janajagruti Samiti Blog Website

A responsive and informative Blog Website designed to showcase articles, updates, and videos related to Dharma awareness, social campaigns, and cultural topics. This project integrates HTML, CSS, JavaScript, and embedded YouTube content for a rich user experience.

🌟 About the Project

This website is a blog-style platform inspired by the work of
Hindu Janajagruti Samiti link :-  https://anand-chauhan.github.io/HJS/

It focuses on:

📰 Publishing blog articles and updates

📢 Sharing awareness campaigns

🎥 Embedding YouTube videos

📚 Promoting knowledge and discussions

The organization conducts activities related to awareness, education, and campaigns around Dharma and society.

✨ Features

📰 Blog listing page with latest posts

📄 Individual blog detail pages

🎥 YouTube video integration

📱 Fully responsive design

🔍 Clean and readable UI

⚡ Fast loading performance

🛠️ Tech Stack

HTML – Page structure

CSS – Styling & layout

JavaScript – Dynamic content & interactions

YouTube Embed API – Video integration
